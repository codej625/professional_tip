# 개인정보 수집과 이용 동의

<br />
<br />

* 어디까지 개인정보일까?

---

```
개인정보 보호법에 따라 특정 개인을 알아볼 수 있는 정보라면 모두 개인정보로 볼 수 있다.

흔히 이름, 연락처, 주민등록번호 등 직접적인 정보만을 생각하기 쉽다.

직접적인 정보뿐만 아니라 다른 개인정보와 결합해서 사람을 식별할 수 있는 간접적인 정보도 개인정보에 해당한다.

예를 들어 직장, 직책, 혈액형, 종교 모두 개인정보에 해당한다.
```

1. 수집 시 반드시 포함해야 할 3요소

```
개인정보를 수집할 때는 아래 3가지 항목을 구체적으로 명시해야 하며,
응답자가 동의 여부를 자유롭게 선택할 수 있도록 설계해야 한다.
```

<br />

`수집 목적` 

```
어떤 목적으로 개인정보를 수집하는지

ex) 이벤트 경품 발송
```

<br />

`수집 항목` 

```
이름, 연락처 등 구체적인 항목

ex) "등"과 같은 모호한 표현 지양
```

<br />

`보유 및 이용 기간` 

```
구체적인 파기 시점

ex) 입력일로부터 6개월, 목적 달성 시 즉시 등
```

<br />

```
- 동의를 거부할 권리가 있다는 사실과,
  거부 시 불이익(경품 제외 등)도 함께 안내해야 한다.
```

<br />
<br />
<br />

2. 위탁 vs 제3자 제공 구분하기

```
개인정보가 외부로 전달될 때,
성격에 따라 고지 방법이 달라진다.
```

<br />

`위탁 (Consent Not Required)`

```
우리 회사의 업무 처리를 위해 외부 툴(스티비, 토스페이먼츠 등)을 사용하는 경우.
별도 동의 대신 개인정보 처리방침 페이지에 "업체명"과 "업무 내용"을 명시하면 된다.
```

<br />

`제3자 제공 (Consent Required)`

```
정보를 받는 업체(제휴사, 보험사 등)의 목적을 위해 정보를 제공하는 경우.
반드시 고객에게 별도의 동의를 받아야 하며 업체명, 목적, 항목, 기간을 명시해야 한다.
```

3. 동의 문항 작성 예시

```
몇 가지 예시
설문/이벤트용, 수강생 모집용
```

<br />

`설문/이벤트용`

```
[개인정보 수집 및 이용 동의]

- 수집하는 자: (주)OOO
- 수집 목적: 설문 참여자 보상 및 리워드 발송
- 수집 항목: 이름, 휴대폰 번호
- 보유 기간: 수집 일로부터 30일 이내 (보상 발송 완료 후 파기)

※ 귀하는 동의를 거부할 권리가 있으나, 거부 시 이벤트 참여가 제한될 수 있습니다.
( ) 동의함  ( ) 동의하지 않음
```

<br />

`수강생 모집용`

```
[개인정보 수집 및 이용 동의]

가. 수집하는 개인정보 항목 및 수집방법
  - 수집항목: 이름, 지역, 상호, 휴대폰번호
  - 수집방법: 홈페이지

나. 개인정보의 수집 및 이용목적
  - 상담 신청에 따른 본인 확인 및 본인 의사 확인
  - 담당자의 안내 전화 및 서비스 상세 설명

다. 개인정보의 보유 및 이용기간
  - 회사는 개인정보 수집 및 이용목적이 달성된 후에는 해당 정보를 지체 없이 파기합니다.

라. 동의 거부 권리 및 불이익 고지
  - 귀하는 개인정보 수집 및 이용에 동의를 거부할 권리가 있습니다. 
    단, 동의 거부 시 상담 신청 및 서비스 안내가 제한될 수 있습니다.

( ) 동의함  ( ) 동의하지 않음
```

```
개인정보의 보유 및 이용기간 같은 경우는
기간에 따라 "동의일로부터 1년간" 이런식으로 변경 가능
```
