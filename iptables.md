# iptables

<br />
<br />

* 리눅스의 방화벽 및 네트워크 패킷 필터링 도구

---

```
iptables는 간단히 말하면,
리눅스 시스템으로 들어오고 나가는 네트워크 트래픽을 제어하는 프로그램이다.

컴퓨터의 문지기 역할을 한다고 보면된다.

최근에는 nftables라는 후속 기술로 대체되고 있지만,
여전히 많은 시스템에서 서버 관리나 네트워크 보안에서 필수적인 도구로 사용중이다.
```

<br />
<br />
<br />
<br />

1. 주요 용도

| 기능                              | 설명                                              |
|----------------------------------|--------------------------------------------------|
| 방화벽 구축                         | 특정 IP나 포트로부터의 접속을 차단하거나 허용              |
| NAT(Network Address Translation) | 내부 네트워크와 외부 인터넷 간의 주소 변환                 |
| 포트 포워딩                         | 특정 포트로 들어온 트래픽을 다른 곳으로 전달               |
| 패킷 필터링                         | 네트워크 패킷의 출처, 목적지, 프로토콜 등을 기준으로 허용/차단 |

<br />
<br />
<br />

2. 간단한 예시

```zsh
# 22번 포트(SSH) 접속 허용
iptables -A INPUT -p tcp --dport 22 -j ACCEPT

# 특정 IP 차단
iptables -A INPUT -s 192.168.1.100 -j DROP

# 모든 HTTP(80포트) 트래픽 허용
iptables -A INPUT -p tcp --dport 80 -j ACCEPT
```
