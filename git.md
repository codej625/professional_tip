# 깃 커밋 혹은 커밋 메시지 변경하기

<br />
<br />

* 간혹 커밋이나 커밋 메시지를 실수할 때가 있다.
---

```
급하게 업무를 하다 보면 하지 말아야 할 커밋이나
커밋 메시지를 이상하게 적고 커밋하는 경우가 생긴다.

이런 상황에서 간단하게 변경하는 방법을 알아보자.
```

<br />
<br />
<br />
<br />

1. 커밋 되돌리기

```
// 가장 최근 커밋을 취소하지만, 작업한 내용은 그대로 유지한다.

git reset --soft HEAD~1
```

<br />
<br />
<br />

2. 커밋 메시지 되돌리기

```
// 가장 최근의 메시지 수정하기

git commit --amend
```
