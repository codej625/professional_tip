# Tailwind 4

<br />
<br />

* 멀쩡한 CSS 문법을 두고 왜 테일윈드를 사용할까?

---

<br />

`클래스 이름 짓기 없음`

```
.card-inner-wrapper,
.section-title-secondary 같은 이름을 안 만들어도 됨
```

<br />

`파일 왔다 갔다 안 함`

```
className에 바로 스타일 나열해서,
컴포넌트만 봐도 스타일이 보임
```

<br />

`값이 통일됨`

```
mt-4, text-sm처럼 디자인 토큰이 정해져 있어서,
팀 전체가 비슷한 간격·크기를 쓰기 쉬움
```

<br />

`반응형/다크`

```
sm:, dark: 같은 클래스 이름만으로
반응형, 다크모드 적용되어 쓰기 편함.
```

1. 레이아웃

| 클래스 | 의미 | 예시 용도 |
|--------|------|-----------|
| `flex` | display: flex | 가로/세로 정렬 |
| `flex-col` | flex-direction: column | 세로 배치 |
| `flex-1` | flex: {1 1 0%} | 남는 공간 채우기 (main 등) |
| `flex-wrap` | flex-wrap: wrap | 줄바꿈 허용 |
| `items-center` | align-items: center | 세로 중앙 정렬 |
| `justify-between` | justify-content: space-between | 양끝 정렬 |
| `justify-center` | justify-content: center | 가로 중앙 정렬 |
| `gap-2` | gap: {0.5rem} | 자식 사이 간격 |
| `gap-x-2` | column-gap: {0.5rem} | 가로만 간격 |
| `w-full` | width: {100%} | 가로 꽉 참 |
| `w-56` | width: {14rem} | 고정 너비 (사이드바 등) |
| `max-w-[1080px]` | max-width: {1080px} | 최대 너비 제한 |
| `max-w-[1200px]` | max-width: {1200px} | 최대 너비 제한 |
| `mx-auto` | margin-left/right: {auto} | 가로 중앙(블록) |
| `min-h-screen` | min-height: {100vh} | 화면 높이만큼 최소 |
| `min-h-full` | min-height: {100%} | 부모 높이만큼 최소 |

2. 여백 (Margin / Padding)

```
숫자는 4px 단위라고 보면 된다.

1 = {4px}
2 = {8px}
3 = {12px}
4 = {16px}
6 = {24px}
8 = {32px}
```

| 클래스 | 의미 |
|--------|------|
| `mt-2` | margin-top: {8px} |
| `mt-3` | margin-top: {12px} |
| `mt-4` | margin-top: {16px} |
| `mt-6` | margin-top: {24px} |
| `mt-8` | margin-top: {32px} |
| `mb-2` | margin-bottom: {8px} |
| `mb-4` | margin-bottom: {16px} |
| `px-4` | padding-left/right: {16px} |
| `px-3` | padding-left/right: {12px} |
| `py-2` | padding-top/bottom: {8px} |
| `py-6` | padding-top/bottom: {24px} |
| `py-10` | padding-top/bottom: {40px} |
| `p-4` | padding: {16px} |
| `p-6` | padding: {24px} |
| `pt-6 pb-4` | 위 {24px}, 아래 {16px} |

3. 글자 (폰트 / 크기 / 색)

| 클래스 | 의미 |
|--------|------|
| `text-xs` | {12px}, 작은 보조 텍스트 |
| `text-sm` | {14px}, 본문·캡션 |
| `text-base` | {16px}, 기본 본문 |
| `text-xl` | {20px}, 제목 |
| `text-2xl` | {24px}, 큰 제목 |
| `font-medium` | font-weight: {500} |
| `font-semibold` | font-weight: {600} |
| `font-bold` | font-weight: {700} |
| `leading-relaxed` | line-height: {1.625} |
| `not-italic` | font-style: normal (address용) |
| `text-center` | text-align: center |
| `underline` | text-decoration: underline (링크) |

<br />


`색 (zinc = 회색 계열)`

| 클래스 | 의미 |
|--------|------|
| `text-zinc-500` | 연한 회색 (보조) |
| `text-zinc-600` | 본문용 회색 |
| `text-zinc-700` | 조금 더 진한 회색 |
| `text-zinc-900` | 거의 검정 (제목) |
| `dark:text-zinc-400` | 다크 모드에서 밝은 회색 |
| `dark:text-zinc-100` | 다크 모드 제목용 |

4. 배경 / 테두리

| 클래스 | 의미 |
|--------|------|
| `bg-background` | CSS 변수 --background (globals.css) |
| `text-foreground` | CSS 변수 --foreground |
| `bg-white` | 배경 흰색 |
| `bg-zinc-50` | 연한 회색 배경 (카드 등) |
| `bg-zinc-100` | 조금 더 진한 연회색 |
| `dark:bg-zinc-800` | 다크 모드 배경 |
| `dark:bg-zinc-800/50` | 다크 모드 반투명 |
| `dark:bg-zinc-900` | 다크 모드 더 어두운 배경 |
| `dark:bg-zinc-950` | 다크 모드 가장 어두운 배경 |
| `border` | {1px} 테두리 |
| `border-t` | 위쪽만 테두리 |
| `border-b` | 아래쪽만 테두리 |
| `border-zinc-200` | 테두리 색 (라이트) |
| `dark:border-zinc-700` | 다크 모드 테두리 |
| `dark:border-zinc-800` | 다크 모드 테두리 (더 어두움) |
| `rounded` | border-radius 작게 |
| `rounded-lg` | border-radius 조금 더 크게 |
| `rounded-xl` | border-radius 크게 |
